Här finner du ett riktigt kasst api.

Såhär gör du typ:

Skicka en liten POST med:
  • type = [namn på bad_Script]
  • content = [vad scriptet tar emot för något]
lista över script finner du på https://github.com/Gnidleif/bad_scripts

Exempel:
curl -d "type=sverjeven&content=Sluta förstöra vårt fina land, jag älskar sverige" https://lekskola.com/bad_api
sluta FÖRSTÖRA vårt fina Land,,, jag älskar SVERGE

curl -d "type=spongebob&content=hähähä tänk om jordnötter skulle växa i jorden, vad knäppt det vore" https://lekskola.com/bad_api
